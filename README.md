
# Cooking Assistant

A simple Python script that uses a Large Language Model (LLM) to search a local database of recipes organized by categories.

## Features

- Search recipes by category
- Natural language queries powered by LLM
- Local database for offline access


## Requirements

- Python 3.13+
- Required packages: `openai`


## Installation

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python cooking_assistant.py`


## Usage

Run the script and enter your query, e.g., "Show me Italian pasta recipes."  


## Database

Recipes are stored locally in a database, categorized for easy retrieval.
